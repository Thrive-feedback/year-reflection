# Year Reflection Project

## Overview
This is a year reflection application built with React and Vite.

## Project Structure
- `/src` - Source code
- `/public` - Static assets
- `node_modules/` - Dependencies

## Technologies
- React
- Vite
- Tailwind CSS (likely, based on common setup)
- Radix UI components
- Lucide React (icons)

## Development
- Development server: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
